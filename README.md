# WiFiGuard

WiFiGuard is a Python-based Wi-Fi security monitoring application developed using Tkinter.

## Features

- Detects the currently connected Wi-Fi network
- Supports trusted network configuration
- Displays Trusted / Untrusted status
- Generates activity logs
- Saves logs to wifilog.txt
- User-friendly graphical interface

## Technologies Used

- Python
- Tkinter
- Subprocess
- File Handling

## How to Run

1. Install Python 3
2. Download the project files
3. Run:

python wifiguard_gci.py
